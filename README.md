# Rithea's Portfolio

A personal portfolio website built with **React.js** and **JavaScript**, showcasing my skills, projects, and experience as a frontend developer.



## Preview

<img width="944" height="437" alt="image" src="https://github.com/user-attachments/assets/b60e0332-f541-477b-b592-8f076f8ed302" />


##  Features

-  **Dark / Light mode** toggle with persistent theme (saved in localStorage)
-  **Animated typing effect** cycling through roles (Frontend Developer, React Native Developer, etc.)
-  **Fully responsive** — works on mobile, tablet, and desktop
-  **Skills section** with animated progress bars
-  **Education & experience timeline**
-  **Projects showcase** with live demo & GitHub links
-  **Contact section** with email, LinkedIn, GitHub, and social links
-  **Downloadable CV/Resume**
-  Animated starfield background

##  Built With

- **React.js** — UI library
- **JavaScript (ES6+)**
- **CSS3** (custom properties for theming, animations, flex/grid layouts)
- **Tailwind CSS** — utility classes
- **Vite** — build tool / dev server

## Getting Started
**1. Clone the repository**
```bash
git clone https://https://github.com/chhaysokrithea-cloud/portfolio-reactjs.git
cd portfolio-reactjs
```

**2. Install dependencies**
```bash
npm install
```

**3. Run the development server**
```bash
npm run dev
```

**4. Build for production**
```bash
npm run build
```
